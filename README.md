# Protego

A stupidly simple plugin that prevents projectiles and explosions from breaking hanging entities.

As of [Protego 1.2](https://github.com/KadTheHunter/Protego/releases/tag/1.2), it also includes the `/evanesco` command, which detects and removes Armor Stands that have negative Health / Death Time values.

Name inspired by the Shield Charm from Harry Potter.
Evanesco command inspired by the Vanishing Spell from Harry Potter.
